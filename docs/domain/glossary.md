# Глоссарий доменной модели

| Термин | Определение |
| --- | --- |
| Пользователь | Человек, который создаёт и управляет проектами в системе. |
| Команда | Группа пользователей, объединённых для совместной работы над проектами. |
| Проект | Единица планирования, содержащая задачи, сроки и связанные артефакты. |
| Задача | Конкретная работа в рамках проекта, имеющая исполнителя, статус и дедлайн. |
| Статус задачи | Текущее состояние задачи (новая, в работе, завершена, заблокирована). |
| Комментарий | Текстовое сообщение внутри задачи для уточнений и истории решений. |
| Вложение | Файл или ссылка, прикреплённые к задаче или проекту. |
| Эпик | Крупная функциональная область, объединяющая связанные задачи и проекты. |
| Метка | Ключевое слово для категоризации задач и проектов. |
| Релиз | Запланированная поставка функциональности или набора задач. |
