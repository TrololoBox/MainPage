# Доменная модель

Подготовлена и утверждена базовая доменная модель для управления проектами и задачами. Ключевые сущности: Пользователь, Команда, Проект, Задача, Комментарий, Вложение, Эпик, Метка и Релиз.

- Глоссарий терминов: [glossary.md](./glossary.md)
- Исходник ER-диаграммы (Mermaid): [er.mmd](./er.mmd)

Диаграмма отражает связи создания проектов пользователями, владения проектами командами, декомпозиции проектов на задачи, назначения исполнителей, добавления комментариев и вложений, а также группировки задач эпиками, метками и релизами. Модель прошла внутренний ревью и согласована для дальнейшей детализации и реализации.
